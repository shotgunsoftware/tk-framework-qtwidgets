Shotgun Global Search Widget
#############################################

The QT global search widget is similar to the global search
found in Shotgun. It returns a number of matches based on a text
that is being typed in by the user.

.. image:: images/global_search.png

Once a user selects an object, a signal fires to indicate the request.
